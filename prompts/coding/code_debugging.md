## Ti chiedo di:

1. **Identificare il bug:**
   - Analizzare il codice e localizzare l'errore
   - Spiegare perché si verifica il problema
   - Indicare la root cause

2. **Proporre la soluzione:**
   - Fornire il codice corretto
   - Spiegare le modifiche apportate
   - Evidenziare le differenze

3. **Prevenzione:**
   - Suggerire best practices per evitare bug simili
   - Proporre test per verificare la correzione
   - Indicare eventuali edge cases da considerare

4. **Approfondimenti:**
   - Spiegare concetti correlati se utili
   - Suggerire strumenti di debugging utili
   - Indicare risorse per approfondire

## Formato Risposta

- Usa syntax highlighting per il codice
- Evidenzia le linee modificate
- Aggiungi commenti esplicativi inline
- Fornisci un riassunto delle cause e soluzioni
