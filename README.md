# Prompts e AI

Questa repository contiene una collezione di prompt generici e esempi per interagire efficacemente con modelli di linguaggio (LLM).

## Struttura

```
prompts_and_ai/
├── templates/           # Template generici riutilizzabili
│   ├── code_review.md       # Template per revisione codice
│   ├── summarization.md     # Template per riassunti
│   ├── translation.md       # Template per traduzioni
│   ├── brainstorming.md     # Template per brainstorming creativo
│   └── email_writing.md     # Template per scrittura email professionali
├── prompt_1.md          # Esempio: Configurazione Claude Desktop + WSL
├── prompts_2.md         # Esempio: Test
├── prompt_3_data_analysis.md     # Esempio: Data Analysis con Python
└── prompt_4_api_documentation.md # Esempio: Documentazione API
```

## Come Usare i Template

I template nella cartella `templates/` sono progettati per essere copiati e personalizzati secondo le tue esigenze. Ogni template include:

- Istruzioni per l'AI
- Sezioni da compilare con i tuoi dati
- Suggerimenti per ottenere risultati migliori

## Esempi

Gli esempi (`prompt_*.md`) mostrano casi d'uso reali e dettagliati che possono servire come ispirazione per creare i tuoi prompt personalizzati.

## Contribuire

Sentiti libero di aggiungere nuovi template o esempi seguendo lo stile esistente.
