Questa repository contiene una collezione di prompt generici e esempi per interagire efficacemente con modelli di linguaggio (LLM).

```

│   ├── translation.md          # Template per traduzioni
│   ├── brainstorming.md        # Template per brainstorming creativo
│   └── email_writing.md        # Template per scrittura email professionali
│
├── prompts/                # Prompt organizzati per categoria
│   ├── coding/                 # Prompt per sviluppo software
│   │   ├── api_documentation.md    # Documentazione API REST

│   │
│   ├── data/                   # Prompt per analisi dati

│   │
```

## Come Usare i Prompt

### Template Generici
I template nella cartella `templates/` sono progettati per essere copiati e personalizzati secondo le tue esigenze. Ogni template include:

- Istruzioni per l'AI
- Sezioni da compilare con i tuoi dati
- Suggerimenti per ottenere risultati migliori

### Prompt per Categoria
I prompt nella cartella `prompts/` sono organizzati per campo di applicazione:

- **coding/** - Prompt per sviluppo software, debugging, documentazione
- **data/** - Prompt per analisi dati, visualizzazione, data science
- **devops/** - Prompt per CI/CD, infrastruttura, configurazione ambienti
- **finance/** - Prompt per analisi finanziaria, valutazione, investimenti
- **math/** - Prompt per risoluzione problemi matematici, calcolo, statistica

## Esempi

Gli esempi mostrano casi d'uso reali e dettagliati che possono servire come ispirazione per creare i tuoi prompt personalizzati.

## Contribuire

Sentiti libero di aggiungere nuovi template o prompt seguendo lo stile esistente. Per i nuovi prompt, scegli la categoria appropriata nella cartella `prompts/` o creane una nuova se necessario.
